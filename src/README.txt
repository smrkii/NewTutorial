* Potegni lwjgl: https://osdn.net/projects/sfnet_java-game-lib/downloads/Official%20Releases/LWJGL%202.9.3/lwjgl-2.9.3.zip/
V podanem posnetku potegne en drug lwjgl zip.
* Nastavi okolje preko tega videa --> https://www.youtube.com/watch?v=Jdkq-aSFEA0
* V intelij klikni File --> Project Structure --> Dependencies --> klikneš na + in dodaš lwjgl.jar, lwjgl_util.jar ter slick-util.jar
* V direktorij od jave(jst mam jdk-12.0.2) katero uporabljaš v intelij-u, ponavad se nahaja v C:\Program Files\Java\jdk-12.0.2
v file bin dodej vse ddl fajle kateri se nahajajo v zipu od lwjgl-ja katerega si podegnil. Ti ddl-ji se nahajajo v direktoriju native,
izbrat morš pa tiste ddl-je odvisno od operacijskega sistema. Če maš windows pol koperaš ddl-je iz windows datoteke v javo kot je
opisano zgori.
